# S1S - Seja 1 Sujeito (Be a Subject)

## Overview
S1S is a web-based platform designed to connect researchers with potential research subjects. The system allows researchers to register their studies and find suitable participants, while individuals can sign up to participate in various research projects.

## Academic Context
This project was developed as part of the **Projetos** (Projects) course within the **Desenvolvimento de Sistemas** (Systems Development) program.

## Features
- **User Authentication**: Login system for both Researchers and Subjects.
- **Researcher Registration**: Dedicated registration form for researchers including institution and area of study details.
- **Subject Registration**: Registration for individuals willing to participate in research.
- **Database Integration**: Built with PHP and MySQL for data management.

## Technologies Used
- **Backend**: PHP
- **Frontend**: HTML, CSS, JavaScript (Bootstrap)
- **Database**: MySQL

## Setup
1. Configure the database connection settings (currently located in files like `index.php` and `includes/Banco.php`).
2. Run the application on a PHP-enabled server (e.g., Apache, Nginx).
3. Ensure the MySQL database `S1S_005` is created and accessible.
